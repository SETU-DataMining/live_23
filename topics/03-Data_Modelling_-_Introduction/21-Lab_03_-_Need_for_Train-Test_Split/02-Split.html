Split Data into train and test

The sklearn is python's main library for performing machine learning tasks, including preprocessing and model evaluation steps.

First we import the default function to split data

1
from sklearn.model_selection import train_test_split

and we use it as follows

1
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.40, random_state=42)
  • The function takes in both the features, X, and the target, y, and returns four arrays which we typically denote by X_train, X_test, y_train, and y_test respectively.
  • The parameter test_size specifies the proportion of the data that will be retained for test/evaluation.
  • The parameter random_state is used so that we get a random split but the split is reproducible. Changing the value of 42 will result in a different split.
