Association Analysis

This week lab consists of two notebooks using association analysis, firstly using brute force and then the second, using python library mlxtend to apply more scalable algorithms to larger data sets.

The notebooks can be obtained from the following links:

  • Using brute force techniques for association analysis
  • Data investigation and association analysis using the classic groceries data set

Installation

You will need to update your python packages by installing mlxtend using the following command:

1
    conda install -c conda-forge mlxtend

Resources

The datafile came from the R coimmunity and a good source is "Machine Learning with R" - datasets groceries.csv.
