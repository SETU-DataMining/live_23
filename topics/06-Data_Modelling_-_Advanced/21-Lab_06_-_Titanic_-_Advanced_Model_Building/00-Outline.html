Titanic - Advanced Model Building

Aim

In this week's lab we will apply the advanced modelling techniques covered in the lecture to develop a model for the Titanic dataset. This dataset has been studied extensively and there are many resources online to help you.

By end of practical we should have a notebook with the following structure:

  1. Load Dataset and Initial Clean
  2. Baseline Model
  3. Feature Engineering
  4. Hyper-parameter Tuning
  5. Tuned Model
  6. Test/Evaluate Model

In each of the above sections we have written outline code and suggestions for how you should approach this task. However, we strongly encourage you to try your own ideas.

We are going to limit ourselves to the three classifiers that we covered in week 6:

  • LogisticRegression
  • K Nearest Neighbors
  • Bernoulli Naive Bayes

and the ensemble learner

  • AdaBoostClassifier

(We won't use the ensemble learner RandomForestClassifier which is based on decision trees (to be covered in week 8).)
