Software Installation

As mentioned before, the Monday lab sessions are usually based on the material covered in the preceding Friday's lecture. However, the first lecture was introductory so this is a good opportunity to install the software we will be using in the rest of the module.

Python will be used in this module. We will also use Jupyter notebooks to write and run data mining workflows, because they are ideal for investigating data, by freely mixing code, its outputs (both in text and graphical format) and discussion.
