Pre-Start Lab

Typically the Monday lab sessions are based on the material covered in the preceding Friday's lecture. However, for the first week we will run a more informal (and hopefully relaxed) session where you are asked to examine a small dataset using whatever tools you prefer. We want you to focus on the data, and what questions should one ask when given data. Later (next week) we will worry about software and learning new tools (python, pandas, seaborn, scikit-learn, etc).

The aims of this lab session are:

  • Review of Basic Data Exploration Skills.

    We (or more accurately you) will take a small dataset based on records of server tips, and explore it using the software tool of your choice. Later we will preform the same analysis using pandas, but for now the focus is on seeing what you can see when you look at data.

  • Develop familiarity with Moodle, in particular Moodle activities (Quiz)

    To help direct your data exploration, mentioned above, we have a small Moodle quiz based on the server tips. This quiz is formative (i.e., not summative) so does not contribute towards your overall grade. However, it is a good opportunity gain familiarity with Moodle, Moodle quizzes & assignments.
