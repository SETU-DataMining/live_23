Natural Language Processing: NLTK and sklearn

As usual, each notebook should be downloaded to your laptop and run using jupyter-notebook or jupyter-lab, as described in previous practicals.

The notebooks can be obtained from the following links:

  • Using NLTK to process a variety of documents
  • Using sklearn TF-IDF to compare inaugural speeches

Installation

You will need to update your python packages by installing nltk and altair, using the following commands:

1
2
3
conda install -c anaconda nltk

conda install -c conda-forge altair

Resources

The NLTK package comes with optional document corpora, which are needed for the NLTK notebook.

For the sklearn TF-IDF notebook, you will need to download the inaugural speeches data from kaggle, or alternatively you can just download them from here.
