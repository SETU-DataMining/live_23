Feature Engineering

Aim

In this section we want to apply our various feature engineering techniques to the data. Taking the list of techniques given in class

and applying to the Titanic dataset we have

Feature Engineering

Imputation

  • Age --- use the mean/median. But create binary indicator feature to record location of NA.
  • Embarked --- few (3) NA, feature has large modal level => use the mode.
  • Cabin --- create binary indicator feature to record location od NA.
  • Fare --- create binary indicator feature to record location od NA.

Outlier

  • 'Fare' --- I would use consider Censoring. The zero values, I would treat as NA.

Binning

  • Age -- Think of binning based on babies, children, teenagers, adults, elderly.
  • Fare -- I have not used binning here but you could.

Log Transform

  • Fare is crying out for a log transformation, but you need to deal with the zero values first.

Categorical Encoding

  • We went for one-hot encoding for the initial categorical features. This is good for now.
  • Title --- combine rare (<10) levels in misc

Normalisation and Scaling

  • We went for standard scaling (mean=0, std=1) for the numerical features. This is probably best if we have sorted out Fare and its outliers/high skew. Or you could try RobustScaler.

Grouping Operations

  • Can't think of any, right now, does not mean none are there.

Splitting

  • Title --- We could extracte the title from the Name column (see below)
  • Deck --- We could extracte the deck information from the Cabin column

Feature Transformation

There are a number of plausible transformation that you could do, for example

  • Family_Size --- equal to SibSp + Parch

Feature Selection

Drop features that we now longer need because we probably have extracted all of the information we can from them: Name, Cabin etc.

In a bigger dataset we could perform filter selection using

  • A filter such as SelectKBest which selects features based on some heuristic (correlation,chi2,mutual information, etc.) to include in the model.
  • A wrapper such as RFECV which selects features based on their impact on the model.

See scikit's Feature selection documentation for more details.
