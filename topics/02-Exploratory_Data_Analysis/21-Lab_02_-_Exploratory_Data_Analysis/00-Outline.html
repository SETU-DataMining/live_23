Outline

Aim

This week's lecture outlines the steps used to perform an exploratory data analysis of three datasets. The aim of this lab is to reproduce the generated metrics and visualisations presented in lecture slides.

In addition to covering technical questions/issues, we would hope that the lab session on Monday would also be used to discuss interpretation of the generated metrics and visualisations. (In the lecture I sometimes picked extreme cases to illustrate a point, for example, my choice of the chemical features in the Algae Bloom dataset to decide if the distribution was normal or not.)

Copies of the three datasets given in this ZIP archive.

First Pass — Load and Initial Clean

  • Nearly all of the steps for this were given in the lecture notes. Only steps missing were converting some of the categorical features with type object to category (e.g. Embarked in the Titanic dataset) and saving the resulting dataset out to pickle format so that meta information (type) is preserved (Only did this for Algae Blooms dataset.)

Note that copying and pasting code from the slides can introduce problems due to the typesetting of certain characters (such as quotes and the minus sign). Instead of copying and pasting you should download archive of code snippets where each snippet is in a separate txt file.

Second Pass — Target and Individual Features

  • Here we presented only a subset of the features in each dataset. For example in the Algae Blooms dataset, of the 8 chemical level features we only looked at max_pH and mean_Chlor.

Third Pass — Relationship between Features and Target

  • Reproduce the visualisations given in the slides.

Extra Tasks

In addition to the above, the following is some additional (optional) tasks:

  • Resolving 0 values in feature Fare in Titanic dataset.
    Using the value of the other features in the observations where Fare=0 determine an appropriate strategy to resolve these missing values.

  • Generating the proposed feature Title from feature Name in Titanic dataset.
    This is very much a (python) programming, rather than a pandas/data mining task. So would suggest you skip this for now if you are new to python.
