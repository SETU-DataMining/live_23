Outline

During our Topic-03 lecture we covered the difference in approach between statistical models and machine learning models. In particular, the need to split the data into two sets — a train set to be used for fitting and a test set to be used for evaluation — when following the machine learning approach to modeling.

In advance of the next topic, by Bernard, on regression model and its statistical analysis, we will look at another example, illustrating the need for the train/test split when following the machine learning approach.

Again: you should reflect on our goal — we want to build a prediction model, so our metrics to evaluate our model must be based on how well the model predicts on unseen data.

In the following, we will:

  • Generate a simple noisy dataset
  • Split the data into train and test
  • Create polynomial features
  • Fit/train model
  • Compute the Mean Squared Error (MSE)

We want you to follow the steps provided and generate all of the given results/graphs.
