Module Overview

Welcome to Data Mining,
We will start this module with a warm up lab to set up software and some basic data analysis. The formal lecture will give an overview of the module — what will be covered, how it will be delivered and assessed. In addition, background concepts in statistics and optimisation are discussed.

  • Pre-Start Lab

    Typically the Monday lab sessions will based on the material covered in the preceding Friday's lecture. However, for the first week we will run a more informal (and hopefully relaxed) session where you are asked to examine a small dataset using whatever tools you prefer. We want you to focus on the data, and what questions should one ask when given data. Later (next week) we will worry about software and learning new tools (python, pandas, seaborn, scikit-learn, etc). The aims of this lab session are:
    • Review of existing basic data exploration skills.
    • Develop familiarity with Moodle, in particular Moodle activities (Quiz)

    For this lab, we ask students to install software used in this module (Anaconda (python 3.9 64-Bit)). And gain familiarity with the jupyter-lab environment, that we will use during this module. The aims of this lab session are:
    • Download and install the Anaconda distribution. This includes the python interpretor and standard libraries, a package manager called conda and editors/interfaces for developing in python.
